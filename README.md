# Portfolio

Personal portfolio site for Chibuikem Emeka-Nwuba — CS student at the University of Saskatchewan.

**Live:** https://portfolio-topaz-seven-39.vercel.app

## What's on it

- Featured projects (HandMix, Point B_ank, CodeTalk, Eco-Waste Scheduler, Finance Dashboard)
- Skills, experience, and contact info

## Stack

- Next.js (App Router) + TypeScript
- Deployed on Vercel

## Running locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

Site content (projects, skills, experience) lives in `app/data.ts` — edit that file to update the site.
